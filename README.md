# Automated Sales Dashboard | Excel, Power Query, Data Model

## Project Overview

This project is an automated Excel sales dashboard created to analyze sales performance across multiple years. The main objective of this project was to reduce manual reporting work by using Power Query’s folder-based import method, where new sales files can be added to a source folder and the dashboard can be refreshed automatically.

This project demonstrates practical skills in Excel automation, Power Query data transformation, Data Model usage, PivotTables, dashboard creation, and MIS reporting.

## Tools Used

- Microsoft Excel
- Power Query
- Excel Data Model
- PivotTables
- Pivot Charts
- Slicers
- Excel Formulas

## Dataset

The dataset contains 3+ years of sales data stored in multiple workbooks. The data includes sales transactions, customers, products, salespersons, order priority, shipment details, and order dates.

## Key Steps Performed

- Consolidated 3+ years of sales data from multiple Excel workbooks using folder-based Power Query import
- Cleaned and transformed raw data by handling inconsistencies and standardizing formats
- Loaded the final cleaned dataset into Excel’s Data Model for better performance
- Created KPI cards for key business metrics
- Built PivotTables and Pivot Charts for interactive reporting
- Added slicers for year and salesperson to filter the dashboard dynamically
- Automated the reporting workflow so that adding a new workbook and refreshing the dashboard updates the visuals instantly
